Estrutura recomendada (app completo)

Copie/cole. Isso vira um app real em Python com Streamlit + SQLite.

# Estrutura (recomendado)
estoquepro/
  app.py
  db.py              # conexão + schema + helpers
  services.py        # regras: estoque_atual, custo_medio, registrar_compra/venda/ajuste
  reports.py         # queries de relatório + export
  ui.py              # componentes (opcional)
  requirements.txt
  estoquepro.db      # gerado em runtime

# requirements.txt
streamlit>=1.32
pandas>=2.1

# Rodar
# streamlit run app.py
